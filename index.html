Octopusjs is an object composition library. Build classes using traits for multiple inheritance.

Defined traits to be used in object compositions

	_8.t("mytrait", {doSomeStuff:function() { //do some stuff}});

you can also pass objects from other libraries. Add Backbone.Events as a trait.

	_8.t("events", Backbone.Events);

Use spawn to create object classes

	var obj = _8.spawn("mytrait, events", {construct:function() {
	
	}});
